                       
                  WELCOME TO BEARDOG!
                        
    This minimal version is designed for users to
    choose the programs they want installed.
    
    Try Super + F1 for the Beardog keybinds list.
    https://sourceforge.net/p/lilidog/discussion/

    For the live environment:
    username = 'user', and root password = 'live'
    
    To install Beardog, reboot the iso and choose 
    'Start installer'.

    After installation, use the welcome script to
    see other initial setup options.
